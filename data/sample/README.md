## Provenance

All data in this directory is **100% synthetic**. No real players, real teams, or real footage.

- `mock_telemetry.json`: Player positions follow randomized plausible basketball movement patterns seeded with a fixed random seed for reproducibility.
- `sample_clip.mp4`: Generated using OpenCV — colored rectangles on a green background. No real court footage.
